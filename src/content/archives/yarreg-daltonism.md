# Программист из Рыбинска создал мобильное приложение для дальтоников- Яррег - новости Ярославской области

*Это архивная копия статьи. [Оригинал на Ярославская Регион](https://yarreg.ru/n3yud/)*

---

Программист из Рыбинска Антон Ленев разрабатывает приложение EyeLens, которое поможет людям, страдающим дальтонизмом, различать цвета, читать разноцветные надписи на вывесках или упаковках товаров. Сейчас пилотная версия доступна для платформы Windows 8, вскоре программист планирует создать аналогичную программу для Android и iOS.

Приложение представляет собой набор медицинских цветовых фильтров, которые позволят скрывать или выделять на изображении отдельные цвета. Оно запускается в режиме фотокамеры. Для того чтобы определить тот или иной цвет, необходимо прикоснуться к экрану – система назовет цвет предмета, отображаемого фотокамерой.

По словам Антона Ленева, первая версия приложения еще далека от совершенства, однако разработка продолжается. Программист планирует работать с врачами-офтальмологами, чтобы подобрать наиболее подходящие фильтры для самых распространенных заболеваний.
