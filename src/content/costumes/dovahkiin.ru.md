Костюм Довакина из The Elder Scrolls V: Skyrim воссоздаёт культовый образ Драконорождённого. Железный шлем с рогами, меховой плащ и чешуйчатая броня -- всё выполнено с вниманием к деталям игрового оригинала.

Шлем вылеплен из EVA-пены с рогами из термопластика. Доспехи собраны из отдельных чешуйчатых пластин, имитирующих стальную броню из игры. Плащ дополнен натуральным мехом для аутентичного нордического вида.

Костюм отлично подходит для съёмок на природе, где атмосфера холодного севера Скайрима ощущается наиболее естественно.
