Костюм Тора из фильма «Тор: Рагнарёк» (Marvel Studios): сакаарский гладиаторский образ — нагрудник, наручи, поножа, наплечник, шлем с поворотными «ушами», два клинка и молот. Премьера — **Старкон 2018**.

**Нагрудник, наручи и «юбка»** сделаны из **натуральной неокрашенной кожи растительного дубления** (2 мм задняя часть, 3.1–3.5 мм фронтальная) — кожа выбрана из-за фактуры, прочности и удобства создания узора. Узор переносился прорезанием поворотным ножом, а затем делалось тиснение: кожа смачивалась, штампом продавливался рельеф. Покраска — **морилка для дерева (палисандр)**, пропитка **льняным маслом** (которое дополнительно темнит оттенок). Красный узор поверх — акрил, тени — аэрограф с чёрным.

**Наплечник, поножа и шлем** — EVA-пена (5 мм основа + 1–2 мм декоративные полосы). Ножная броня делалась по «скотчевой» выкройке, снятой прямо с ноги; форма задавалась строительным феном. Наплечник — шаблон Камуи с дополнительными формами из бумажного скотча. Шлем собран по CC-выкройкам с YouTube, поворотные «уши» закреплены на отрезках пластиковой трубы и мебельной стяжке. Неровности EVA правились гравером и выравнивались герметиком.

**Покраска EVA** делалась по единой технологии: сначала несколько слоёв **Plastidip / «жидкой резины»** (синяя для брони и наплечника, серебристо-серая для шлема) — она давала близкий к финалу тон и берегла расход краски. Поверх — базовый акрил, плавные переходы аэрографом (для наплечника даже нарисовано «граффити»), финиш — матовый акриловый лак из баллончика.

**Мечи** сделаны по собственным векторам в Inkscape на **лазерно-резанной фанере**: сначала гравировка узора, затем ручная прорезка по шаблону. Рукоятки обмотаны EVA (5 мм) + два слоя **Bubblestar**, узор сформирован стеками для лепки и выровнен финишной шпатлёвкой для дерева. Покраска — грунт, кистевой базовый цвет, аэрограф с синим и чёрным, глянцевый лак. Векторные файлы автор выложил в свободный доступ по ссылке из статьи.

**Пряжка ремня и круги на плечах** — **3D-печать** с пост-обработкой (шкурка + гравер + герметик), затем прозрачный грунт и кистевой бронзовый акрил с тенями аэрографа.

**Штаны** — ткань для штор подходящей текстуры; акрилом нарисован узор и пришиты кожаные наколенники с тиснением, сформованные вытягиванием смоченной кожи.

В этом образе Антон Ленев **победил в номинации «Лучший косплей»** на финале фестиваля «Охтакон» в ТЦ «Охта молл» (Санкт-Петербург, 7 марта 2020). Подробный процесс изготовления с фотографиями каждого этапа — в [авторской статье ВКонтакте](https://vk.com/@m0rg0t-process-sozdaniya-kostuma-tora-iz-tor-ragnerek).
